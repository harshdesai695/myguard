# Admin Screens — Design Specifications

## Admin Home Dashboard (admin_home_screen)
- **Header**: "Admin Dashboard"
- **Stats Grid**: 2×2 grid with animated stat cards
  - Residents (people icon, primary), Guards (security icon, secondary)
  - Open Tickets (ticket icon, warning), Today's Visitors (walk icon, info)

## Admin Manage (admin_manage_screen)
- **Sections**: Society, Security, Community, Vehicles
- **Each Section**: Header label + tile list
- **Tiles**: Icon avatar + title + subtitle + chevron

## Admin Reports (admin_reports_screen)
- **List**: Report category tiles with colored icons
  - Visitor Reports, Guard Patrol, Helpdesk, Amenity, Move-in/out

## Admin Settings (admin_settings_screen)
- **Sections**: General (Notifications, Appearance, Language), Society (Edit, Gate Timing), Account (Privacy, Terms, About)
- **Sign Out**: Red filled button at bottom
